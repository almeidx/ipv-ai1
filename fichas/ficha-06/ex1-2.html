Cascading Style Sheets

Cascading Style Sheets (CSS) é um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web.

Os valores permitidos pelo atributo background-image são a palavra-chave none ou como um valor <image>. Para especificar várias imagens de plano de fundo, forneça vários valores, separados por uma vírgula.
